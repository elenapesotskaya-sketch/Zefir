# Редактирование каталога вкусных букетов

## Система редактирования

Система позволяет редактировать товары **прямо на странице** - без отдельных окон и диалогов. Просто переходите в режим редактирования, и все карточки становятся интерактивными.

## Как начать редактировать

### Шаг 1: Откройте режим редактирования
1. Прокрутите страницу до конца (футер)
2. Нажмите кнопку **"Редактировать каталог"**

### Шаг 2: Введите пароль
- Пароль: `1931`
- Нажмите Enter или кнопку подтверждения

### Шаг 3: Режим редактирования активен!
Все карточки товаров теперь редактируемые и готовы к изменениям.

## Редактирование товара

Для каждой карточки доступны следующие поля:

### Название товара
- Текстовое поле с полным названием
- Пример: "Нежные тюльпаны" или "Набор пастельных цветов"

### Описание
- Текстовая область с описанием товара
- Может быть несколько строк
- Пример: "Натуральный зефир ручной работы"

### Цена (EUR)
- Числовое поле с ценой в евро
- Изначально установлено 50 EUR
- Редактируется в режиме редактирования

### Статус товара
- Выпадающий список с двумя вариантами:
  - **"В наличии"** - показывает зеленый статус ✓
  - **"Закончилось"** - показывает серый статус
- Выбор один из двух вариантов

### Изображение товара
- Нажмите на кнопку загрузки (иконка в левом нижнем углу фото)
- Выберите новое изображение с компьютера
- Система автоматически загружает и сохраняет фото

## Сохранение изменений

- Каждая карточка имеет кнопку **"Сохранить"** в нижней части
- Кнопка активна только если вы что-то изменили в полях
- После нажатия "Сохранить":
  - Изменения отправляются на сервер
  - Система перезагружает каталог
  - Карточка обновляется с новыми данными

## Выход из режима редактирования

1. Закончив все изменения, нажмите кнопку **"Выход из редактирования"** в футере
2. Вернетесь к обычному просмотру каталога с сохраненными изменениями

## Доступные разделы

Система поддерживает редактирование четырех разделов каталога:

1. **Цветы в букете** 
   - Букеты в нежной упаковке с лентой
   - 6 товаров по умолчанию

2. **Цветы в коробке**
   - Круглые и квадратные коробочки
   - 3 товара по умолчанию

3. **Цветы в корзинке-сумочке**
   - Корзинки-сумочки с цветами
   - 3 товара по умолчанию

4. **Наборы цветов**
   - Наборы для украшения десертов и тортов
   - 2 товара по умолчанию

Для каждого товара указывается:
- Стоимость: 50 EUR (редактируется)
- Статус: "В наличии" или "Закончилось" (редактируется)
- Описание и изображение (редактируются)

## Особенности и безопасность

- **Защита паролем** - доступ к редактированию только с правильным паролем
- **Облачное хранилище** - все данные сохраняются в Vercel Blob Storage
- **Защищенные изображения** - файлы загружаются в приватное хранилище
- **Автосохранение** - после нажатия кнопки "Сохранить" данные немедленно сохраняются
- **Отслеживание изменений** - каждое сохранение записывается с датой и временем

## Советы

- Используйте описательные названия для товаров
- Проверьте цену перед сохранением
- Убедитесь, что статус соответствует реальной ситуации
- Качественные изображения улучшают привлекательность товара
- Можно редактировать несколько товаров подряд, затем выйти из режима
